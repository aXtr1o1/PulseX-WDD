D) Typography (PROVEN from website source)

This is WDD’s UI brand font for web experience. 



2) Body typography (confirmed)

font-weight: 400

line-height: ~1.428571

text color: #191919

background: white
Use this as the baseline in PulseX.

3) Secondary fonts (present but not necessarily brand)

Roboto and Roboto Slab appear in Elementor local font CSS (likely defaults / selective usage).
Treat Isidora Sans as the canonical WDD UI font unless WDD gives a brandbook.

PulseX typography rules

Use Isidora Sans for all UI text

Use consistent hierarchy: H1/H2/H3 + body + caption

For Arabic: you need an Arabic-compatible font pairing (RTL-ready) that doesn’t break the WDD Latin font feel.
